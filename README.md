# knapsack
Knapsack problem solver in golang

## Usage
```
$ go run main.go input.txt
```

## About input file
```
3  <-- Number of kind of items
5, 8, 11 <-- Comma seperated each size of items (Allowing whitespace)
10, 20, 30 <-- Comma seperated each value of items (Allowing whitespace)
25  <-- Knapsack capacity
```
